Example for headless Statamic CMS backend to be used for a [Next.js frontend like this one.](https://github.com/puzzlesremixed/statamic-next-site)
# Prerequisities : 
Install the required deps with `composer install` and run the server with `php artisan serve`. Visit `/cp` to open the Statamic Dashboard.
